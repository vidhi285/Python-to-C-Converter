# Python to C Converter

A GUI-based Python application that converts basic Python code into equivalent C code using syntax translation techniques.

---

## Overview

This project is developed using Python and Tkinter to perform basic source-to-source code conversion from Python language to C language. The application accepts Python code as input and generates corresponding C code output automatically.

The project demonstrates concepts related to:

* Compiler Design
* Syntax Translation
* GUI Development
* Parsing and Code Conversion

---

## Technologies Used

### Python

Python is used to implement the conversion logic and overall application functionality.

### Tkinter

Tkinter is Python’s built-in GUI library used to design the graphical user interface for code input and output display.

---

## Features

* User-friendly GUI interface
* Converts Python syntax into C syntax
* Supports:

  * Variable declarations
  * Input statements
  * Print statements
  * If-Else conditions
  * While loops
  * For loops using range()
  * Boolean condition conversion
  * Increment and decrement optimization
  * Comment conversion

---

## Project Files

### `project.py`

Contains the core conversion logic that translates Python code into C code.

### `cdgui.py`

Contains the Tkinter graphical user interface for entering Python code and displaying generated C output.
